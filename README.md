# 2D Кликер

* Платформа: Andorid

* Основные функции: 
  * Черепаха (то, на что нужно кликать)
  * Жуки (очки(получаются за клики, необходимы для покупки предметов))
  * Магазин (покупка предемтов для автогенерации жуков)
